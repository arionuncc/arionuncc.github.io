Personal Website Tutorial

Sign up for a GitHub account here. You will need to verify your email address in order to create your first repository.

For those who have not used Git or GitHub from the command line, I suggest installing GitHub Desktop. The download and directions can be found here.

Follow the GitHub pages tutorial here. Select the "User or organization site" option and "GitHub for Windows" option

Next go to the repository you created on GitHub, and go to the project setting.
Once in setting scroll down to Launch Automatic Page Generator button and click the button.

Scroll down on the first page and click Continue to Layouts. Choose a layout and click publish page.

Sync the project in GitHub Desktop, check the file folder of the project for the generated files, change or modify your new site!

Your site will be located at yourgithubusername.github.io

Add your pdf resume to the project root directory. Add a link to your resume on the page.
